Quick Left Hackfest August 2011

Hackfest Rules/Guidelines

Implement a DSL gem that Wraps One or More APIs

  • Must call out to at least one API
  • Must have at least one test
  • Must expose at least one public method

Examples

  • Provider agnostic video thumbnail gem ( vimeo , youtube )

Room for creativity

  • Combine multiple APIs
  • Do it ASYNC style
